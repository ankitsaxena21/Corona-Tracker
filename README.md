# CoronaVirus-Tracker

CoronaVirus-Tracker is a MERN Stack App which tracks country as well as statewise data from offical Covid-19 India API.

## Installation

Install dependencies.

```bash
npm install
cd client
npm install

```

Run app.

```bash
npm sever
cd client
npm start

```
